---

## 🔍 ¿Qué hacemos?

En situaciones de emergencia, la información crítica (centros de acopio, refugios, puntos médicos) suele estar dispersa, duplicada o desactualizada. Desarrollamos herramientas tecnológicas de código abierto para:

* **Centralización:** Unificar fuentes de ayuda dispersas en una sola plataforma.
* **Verificación:** Filtrar datos erróneos mediante flujos de automatización.
* **Agilidad:** Proveer una web dinámica de consulta rápida para ciudadanos y rescatistas.

## 🛠 Nuestro Stack Tecnológico

Estamos construyendo sobre cimientos modernos y robustos:

* **Frontend:** [Next.js 16](https://nextjs.org/) (React) para una experiencia de usuario rápida y fluida.
* **Backend:** PostgreSQL + GraphQL para una gestión de datos eficiente y flexible.
* **Automatización:** [n8n](https://n8n.io/) para orquestar la ingesta y limpieza de datos.
* **Web Institucional:** [Hugo](https://gohugo.io/) para máxima velocidad y seguridad.

## 🤝 ¿Cómo puedes ayudar?

La tecnología es un medio, pero el impacto lo generamos entre todos. Buscamos:

1. **Desarrolladores (JS/TS, Python, Go):** Para mejorar los scrapers y optimizar el backend.
2. **Científicos de Datos:** Para ayudarnos a normalizar la data de distintas fuentes.
3. **Comunicadores:** Para mantener la web institucional actualizada con información oficial.

### Primeros pasos para colaborar

1. Revisa los **Issues** abiertos en cada repositorio.
2. Lee nuestra guía de contribución en cada repositorio (`CONTRIBUTING.md`).
3. Únete a nuestro canal de comunicación (¡Enlace próximamente!).
